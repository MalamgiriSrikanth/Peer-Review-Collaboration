# Project Contributors

This document outlines the team coordination and roles for the PeerCollab Student Platform.

## Core Team
- **Lead Developer**: Student Developer 1
  - Responsibilities: Backend API, Authentication, Core Logic.
- **Frontend Lead**: Student Developer 2
  - Responsibilities: UI/UX Design, Tailwind CSS Integration, Client-side Validation.
- **QA & Testing**: Student Developer 3
  - Responsibilities: Review Flow Validation, Mock API Testing, Documentation.

## Coordination Workflow
1. **Planning**: Weekly syncs to define component architecture.
2. **Development**: Feature-branching workflow for main pages.
3. **Review**: Peer evaluations conducted via the platform's own Review Hub.
4. **Integration**: Final merge and UI standardization using the corporate-blue theme.

## Design Standards
- Theme: Standard Academic (Corporate Blue / Gray).
- Font: Inter / System Sans-serif.
- Framework: React + Vite + Tailwind CSS.
